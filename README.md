# BizPub Issue Tracker

  This repository tracks data quality issues reported by users of
  [BizPub.ai](https://bizpub.ai) — an autonomous database of business and
  information systems research publications.

  ## About BizPub.ai

  BizPub.ai indexes articles from top-tier business research journals, including
   metadata such as titles, authors, affiliations, abstracts, DOIs, and
  publication dates. Learn more on the [About page](https://bizpub.ai/about).

  ## How Issues Are Tracked

  When a user reports a data quality issue on BizPub.ai, a GitHub issue is
  automatically created in this repository. Each issue includes:

  - The article title and journal
  - The type of problem (e.g., wrong title, missing authors, broken link)
  - Any additional comments from the reporter

  ## How to Report an Issue

  1. Visit any article page on [BizPub.ai](https://bizpub.ai)
  2. Click the **Report Issue** link in the top-right corner of the article
  3. Sign in (if not already signed in)
  4. Select the issue type(s) and add optional comments
  5. Submit — a GitHub issue will be created here automatically

  ## Issue Categories

  - Missing or wrong title
  - Missing or wrong authors
  - Missing or wrong author affiliations
  - Missing or wrong abstract
  - Missing or wrong DOI
  - Missing or wrong publication date
  - Missing or wrong volume/issue/pages
  - Missing or wrong keywords
  - Wrong journal attribution
  - Broken publisher link
  - Broken or missing PDF link
  - Duplicate article
  - Not an article (editorial, erratum, etc.)
